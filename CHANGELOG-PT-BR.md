# Fluxogrammer v0.1.0

## Nota da versão

### Mudanças

- Adição de um menu de configuração;
- Mecânica de alterar tema do projeto (Claro, escuro e padrão);
- Criação de formato de arquivo próprio (.fluxcfg) para salvar configurações;

### Problemas resolvidos

- Salvamento de linhas inexistentes no projeto.

### Adições para próxima versão

- Criptografia para segurança de arquivos;
- Outras configurações para deixar software mais personalizado para todos os gostos;
- Curvatura em 90 graus da linha de percurso.
